# CSIT 337(Internet Computing) Final Project Camera Shop
This is the final project for Internet Computing, Which is a website specific for Fujifilm Camera Users where they can shop Camera Bodies, Lenses,
Uses Bootstrap 5 Framework for HTML and CSS to make the website mobile friendly
For the database the project used MySQL, and PHP.

# Views
## User view
User view is what the user will se which includes the products and descriptions of the website and the buying function
## Admin Pannel
As a requirement for the project there needs to be a "Adminstrator View" which allows those with admin accounts to manage inventory
without the need of interacting with the database using applications such PHPmyadmin
the admin view also has the ability to create admin accounts.
